Projeto 1: Análise Estatística de Dados Educacionais

Baseado no curso: "Introdução à Ciência de Dados – Estatística Essencial (6h)" - Escola Virtual.GOV

Objetivo: Aplicar conceitos de estatística descritiva aprendidos no curso para analisar a relação entre horas de estudo e desempenho acadêmico.
Metodologia:
1. Geração de dados sintéticos educacionais
2. Estatísticas descritivas (média, desvio padrão, correlação)
3. Visualização com ggplot2
4. Análise de relação linear

Tecnologias:
- R 4.3+
- Pacotes: tidyverse, ggplot2

Como executar
```r
source("analise_estatistica.R")
